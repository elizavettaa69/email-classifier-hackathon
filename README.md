# Email Classifier — автоматическая сортировка корпоративной почты 💌
## Описание проекта
Система автоматически читает письма из папки `inbox/`, классифицирует их по заданным категориям и раскладывает по соответствующим папкам в `processed/`.

## Категории писем 📝
| Категория    | Папка          | Что попадает                            |
|--------------|----------------|-----------------------------------------|
| Спам         | `spam/`        | Спам, фишинг, мошенничество             |
| Инцидент     | `incident/`    | Критичные сбои, ошибки 500              |
| Оборудование | `hardware/`    | Проблемы с гарнитурой, принтером и т.д. |
| Программы    | `software/`    | Проблемы с Chrome, Excel, антивирусом   |
| Доступы      | `access/`      | Запросы прав, новый сотрудник           |
| Финансы      | `finance/`     | Счета, акты, договоры                   |
| Рассылки     | `newsletters/` | Дайджесты, отчёты, приглашения          |
| Другое       | `other/`       | Всё, что не подошло                     |

## Настройка категорий 

Ключевые слова для каждой категории вынесены в отдельный файл `config.json`.  
Это позволяет менять правила классификации **без изменения кода**.

### Как добавить новое ключевое слово

1. Откройте файл `config.json` в любом текстовом редакторе (Блокнот, VS Code)
2. Найдите нужную категорию, например `"spam"`
3. Добавьте новое слово в список `"keywords"`
4. Сохраните файл
5. Запустите программу — новые правила применятся автоматически

### Как добавить новую категорию
1. Откройте config.json
2. Добавьте новый объект в список "categories":
```json
{
    "name": "hr",
    "_comment": "Кадровые вопросы",
    "keywords": ["отпуск", "больничный", "зарплата"]
}
```
3. Сохраните файл и запустите программу

### 🔴 Важно: 
Если файл config.json удалить, программа продолжит работать со стандартными правилами.

## Как запустить 🚀

1. Склонируйте репозиторий:
   ```bash
   git clone https://github.com/elizavettaa69/email-classifier-hackathon.git
   cd email-classifier-hackathon
   ```
2. Положите все письма в папку inbox/
3. Запустите скрипт:
   ```bash
   bash run.sh
   ```

4. Результат появится в папке processed/
